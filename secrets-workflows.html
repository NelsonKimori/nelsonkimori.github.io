Secrets & Workflows

 

This project demonstrates how to securely manage secrets inside GitHub Actions workflows.       Instead of hardcoding credentials, I configured GitHub Encrypted Secrets and used them       in CI/CD pipelines.

 

Key Highlights

 
       
  • Used GitHub Secrets to store API keys securely
    •    
  • Configured workflows to fetch secrets dynamically
    •    
  • Demonstrated safe secret injection during pipeline runs
