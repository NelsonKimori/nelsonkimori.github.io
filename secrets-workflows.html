Secrets & Workflows

This project demonstrates how to securely use GitHub Secrets in a CI/CD workflow. The workflow pulls secrets from GitHub’s encrypted storage instead of hardcoding them in the codebase, ensuring sensitive information like API keys and passwords remain protected.

Key Highlights

  • Stored secrets in GitHub → Settings → Secrets → Actions
  • Referenced secrets safely in GitHub Actions workflows
  • Confirmed secret usage without exposing actual values
