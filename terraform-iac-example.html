Terraform IaC Example

This project shows how to use Terraform to provision secure cloud infrastructure. I used Infrastructure as Code (IaC) principles to deploy repeatable, consistent environments.

Key Highlights

  • Wrote Terraform modules for VPC, EC2, and security groups
  • Implemented least privilege security by controlling inbound/outbound rules
  • Automated state management using remote backend (S3 + DynamoDB)
  • Integrated Terraform with CI/CD for continuous delivery of infrastructure
